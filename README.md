# E-commerce_Sales_Analysis

## Overview
Analysis of E-Commerce sales data covering 2023-2025 using Python and Power BI.

## Dataset
- 5000 orders
- 14 columns
- Covers regions: North, East, West, South

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Analysis
- Total Orders, Sales and Profit
- Top Customers by Orders
- Orders by Region
- Orders by Category
- Effect of Discount on Sales
- Sales and Profit by Category
- Monthly Sales Over Time
- Payment Mode Distribution

## Dashboard
Power BI dashboard with 3 pages:
- Overview
- Orders Analysis
- Sales & Profit

## Key Findings
- Total Sales: 533.67M
- Total Profit: 79.71M
- Home Decor is the top category by sales and profit
- North region leads in sales and profit
- All payment modes are equally used (~20% each)
- No clear relationship between discount and sales
- Monthly sales are stable between 20-25M
